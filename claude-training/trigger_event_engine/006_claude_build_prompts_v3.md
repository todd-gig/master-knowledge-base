# Claude Build Prompts v3

## Prompt 1
Implement a pricing engine in Python using the templates in `pricing_engine/`. It must support tiered, subscription, fixed, and hybrid models, calculate floor price, gross margin, contribution margin, and discount impact, and expose an API.

## Prompt 2
Implement a margin optimization module that evaluates product, bundle, channel, and segment profitability, then recommends actions ranked by margin lift potential and risk.

## Prompt 3
Implement multi-agent coordination with:
- supervisor agent
- handoff contracts
- message contracts
- approval gates
- shared execution logs

## Prompt 4
Implement a real-time trigger and event engine using:
- FastAPI webhooks
- normalized events
- queue-driven workers
- retries
- dead letter queue
- event logging

## Prompt 5
Connect the pricing engine, margin optimizer, multi-agent runtime, and trigger engine so opportunity changes can automatically produce recommendation, pricing, proposal, and sync workflows.
