# Claude Training Pack v3

## Purpose
This pack extends the prior template sets with:
- pricing engine templates
- margin optimization layer templates
- agent-to-agent coordination spec
- real-time trigger/event engine spec
- webhook and queue-driven execution patterns

## Intended Use
Use these markdown files to train or guide Claude to:
- design pricing systems
- optimize commercial structure and margin
- coordinate multiple independent agents
- execute real-time workflows triggered by events
- implement queue-based automation safely

## Pack Structure
- `pricing_engine/`
- `margin_optimization/`
- `coordination_spec/`
- `trigger_event_engine/`

## Operating Principle
Default to deterministic business logic first.
Use model-based reasoning to enhance routing, packaging, prioritization, and drafting, not to replace operational controls.
