# Access and Connectivity Playbook

## Objective
Resolve the primary blocker: fragmented access.

## Required Connections
- GitHub repos shared to main Gigaton owner identity
- Google Drive folders shared to main Gigaton owner identity
- Claude project connected to relevant docs/files
- VS Code authenticated with correct GitHub account
- deployment credentials accessible through approved secure process

## Access Audit Output
| Asset | Owner | Current Access | Required Access | Blocker | Next Action |
|---|---|---|---|---|---|

## Rule
Do not build automation that depends on a personal account unless a migration plan exists.
