# Source Provenance Policy

## Rule
Every major claim, memory, decision, or implementation recommendation should trace to one of:
- uploaded source document
- repo file
- user instruction
- meeting transcript/notes
- validated external documentation
- explicit assumption

## Confidence Levels
- 0.95 canonical project doctrine
- 0.85 strongly supported by source material
- 0.70 reasonable inference from multiple sources
- 0.50 assumption requiring validation
- below 0.50 do not treat as decision-grade
