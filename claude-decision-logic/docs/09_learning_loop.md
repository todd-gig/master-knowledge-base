# Learning Loop

## Purpose
Convert decisions into institutional intelligence.

## Post-Execution Review
For every executed decision record:
- expected outcome
- actual outcome
- variance
- root cause
- whether trust should be upgraded or downgraded
- what markdown artifact should be updated

## Update Targets
- first principles registry
- value matrix weights
- trust matrix weights
- execution thresholds
- playbooks
- templates

## Core Rule
A repeated decision class should get easier, faster, and safer over time.
