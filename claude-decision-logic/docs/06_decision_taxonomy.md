# Decision Taxonomy

## Decision Classes
| Class | Description | Default Approval |
|---|---|---|
| D0 Informational | Summary, framing, non-binding guidance | none |
| D1 Reversible Tactical | Low-cost, low-blast-radius actions | system if T3 |
| D2 Operational | Team/process/workflow changes | owner if T2/T3 |
| D3 Financial | Budget, pricing, commitments | human executive |
| D4 Strategic | Market, product, org direction | executive + audit |
| D5 Legal / Ethical | Contracts, policy, sensitive issues | mandatory human |
| D6 Irreversible / High Blast Radius | Major commitments or public moves | mandatory human |

## Time Horizons
- Immediate: 0-7 days
- Near-term: 8-30 days
- Mid-term: 31-180 days
- Long-term: 181+ days

## Reversibility Tags
- R1 easily reversible
- R2 moderately reversible
- R3 costly to reverse
- R4 effectively irreversible
