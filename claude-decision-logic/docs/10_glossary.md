# Glossary

- **Trust Certificate**: a structured authorization state indicating a decision has passed qualification and trust checks.
- **Immediate Action Decision**: a decision class where speed produces advantage and risk is acceptably bounded.
- **Value Matrix**: scoring model for estimating net value creation.
- **Trust Matrix**: scoring model for estimating confidence and delegation rights.
- **Doctrine**: non-negotiable philosophy and operating intent.
- **Decision Object**: normalized record for a proposed action.
- **Blast Radius**: scope of downside if the decision is wrong.
- **Reversibility**: cost and difficulty of undoing an action.
