# Backend Service Scaffold

## Purpose
Authenticated API layer for:
- intake submission
- namespace draft generation
- namespace promotion/write-back
- retrieval calibration runs
- calibration promotion/write-back
- memory lifecycle transitions
- audit logging

## Suggested stack
- Node.js
- Express
- SQLite or Postgres adapter
- JWT or session auth
- Role-based access control

## Run
This scaffold is illustrative and can be adapted to your preferred backend stack.
