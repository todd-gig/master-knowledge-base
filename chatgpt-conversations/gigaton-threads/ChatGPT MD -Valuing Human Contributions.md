4) Canonical scoring function (trainable + auditable)
=====================================================

### Vector score from indicators (evidence-weighted)

For vector  $v$ :

$$
Score_v = \frac{\sum_i (w_i \cdot x_i \cdot r_i)}{\sum_i (w_i \cdot r_i)}
$$
*    $x_i$ : normalized indicator value (0–7)
*    $w_i$ : indicator weight (default or learned)
*    $r_i$ : evidence reliability (0–1)

### Context-weighted value score

$$
Value = \sum_{v \in 7} (W_v^{context} \cdot Score_v)
$$

### Confidence propagation (simple, effective)

$$
Conf_v = 1 - \prod_i (1 - r_i)
$$

(Or learn a calibration model later.)

2) Outcome engine (probabilistic EV)
====================================

Use a scenario model with:

*   baseline distribution of outcomes without the team
*   contribution effects that shift **probability** and/or **magnitude**

### Base EV model

$$
EV = P(\text{success})\cdot V_{\text{success}} + (1-P(\text{success}))\cdot V_{\text{fail}}
$$

Represent each term as a distribution:

*    $P(\text{success}) \sim \text{Beta}(\alpha,\beta)$ 
*    $V_{\text{success}} \sim \text{LogNormal}(\mu,\sigma)$  (or triangular)
*    $V_{\text{fail}} \sim \text{PointMass}(0)$  or small salvage distribution

3) Map human vectors → outcome deltas
=====================================

A. Convert 7-vector score into two levers
-----------------------------------------

Each person i affects:

1.  **Probability lift**:  $\Delta P_i$ 
2.  **Magnitude lift**:  $\Delta M_i$  (multiplier on success valuation)

Suggested mapping (simple and trainable):

### Probability lift driver

$$
\Delta P_i = k_P \cdot \sigma\Big(\sum_v w^{(P)}_v \cdot \tilde{S}_{i,v}\Big)\cdot \tilde{C}_i
$$

### Magnitude lift driver

$$
\Delta M_i = 1 + k_M \cdot \sigma\Big(\sum_v w^{(M)}_v \cdot \tilde{S}_{i,v}\Big)\cdot \tilde{C}_i
$$

Where:

*    $\tilde{S}_{i,v}$  is score normalized to 0–1 (divide by 7)
*    $\tilde{C}_i$  is aggregated confidence (e.g., mean of  $C_{i,v}$ )
*    $\sigma(\cdot)$  is logistic function
*    $k_P, k_M$  are calibration constants (stage-specific)

**Default intuition (priors):**

*   Probability weights  $w^{(P)}$ : VarianceControl, Trust, Information, Time
*   Magnitude weights  $w^{(M)}$ : Multiplication, Scarcity, Information, Energy

B. Interaction effects (synergy/overlap)
----------------------------------------

Real teams are not additive. Add a synergy term:

$$
\Delta EV_{\text{team}} = \sum_i \Delta EV_i + \sum_{i<j} \gamma_{ij}
$$

Compute  $\gamma_{ij}$  from complementarity:

*   high if one has Multiplication + other has Energy/Time
*   negative if redundant (same niche, low marginal)

Minimal implementation:

$$
\gamma_{ij} = k_\gamma \cdot \text{cosine\_sim}(\vec{S}_i,\vec{S}_j)\cdot (-1) \;+\; k'_\gamma \cdot \text{complementarity}(\vec{S}_i,\vec{S}_j)
$$

* * *

4) Monte Carlo simulation (the simulator)
=========================================

For each run r = 1..R:

1.  Sample baseline  $P_r, V_{\text{success},r}, V_{\text{fail},r}$ 
2.  Apply team deltas:
    *    $P'_r = \text{clip}(P_r + \sum_i \Delta P_{i,r} + \sum_{i<j}\Delta P_{ij,r}, 0, 1)$ 
    *    $V'_{\text{success},r} = V_{\text{success},r}\cdot \prod_i \Delta M_{i,r}\cdot \prod_{i<j}\Delta M_{ij,r}$ 
3.  Compute  $EV'_r$ 
4.  Compute each person’s **marginal contribution** using leave-one-out:
    *    $EV'_{r,\setminus i}$  (re-run with i removed or approximate)
    *    $\Delta EV_{i,r} = EV'_r - EV'_{r,\setminus i}$ 

Aggregate:

*    $\mathbb{E}[\Delta EV_i]$ 
*   percentiles (P10/P50/P90)
*   downside contribution (how much they prevent failure)
*   fragility: variance of  $\Delta EV_i$ 

* * *

5) Convert marginal EV → equity allocation
==========================================

A. Raw share
------------

$$
share_i = \frac{\mathbb{E}[\Delta EV_i]}{\sum_j \mathbb{E}[\Delta EV_j]}
$$

B. Apply governance constraints
-------------------------------

*   floors/ceilings by role
*   minimum founder parity constraints (if desired)
*   option pool reserve
*   adjustment for commitment level (hours/term)

Commitment scalar:

$$
share'_i = share_i \cdot \min\Big(1,\frac{\text{FTE}_i}{1.0}\Big)\cdot \text{term\_factor}_i
$$

Renormalize to sum to 1 within scope.

* * *

6) Vesting + performance gating (to reduce regret)
==================================================

Two-layer equity:

1.  **Base equity** (time vesting)
2.  **Earn-out equity** (outcome vesting tied to vectors/outcomes)

Example:

*   70% time-based vesting
*   30% performance vesting based on realized contribution score:
    *   delivery predictability, revenue impact, retention, fundraising outcomes, etc.

This aligns incentives and reduces litigation risk.

      C[v] = asmt.vector_scores[v].confidence

    # Aggregate confidence (simple + robust)
    conf_i = mean(C[v] for v in 7_VECTORS)

    # Engagement scalar
    commit_i = clamp(person.engagement.fte_fraction, 0.0, 1.0)

    per_person[person.person_id] = {
      "S": S, "conf": conf_i, "commit": commit_i,
      "raw_vector_scores": asmt.vector_scores,
      "assessment_id": asmt.assessment_id
    }

  # -------------------------
  # 2) Convert vectors -> probability lift + magnitude lift parameters
  # -------------------------
  kP = outcomes.calibration.kP
  kM = outcomes.calibration.kM
  wP = normalize_to_sum1(outcomes.calibration.wP)
  wM = normalize_to_sum1(outcomes.calibration.wM)

  FOR pid IN per_person:
    S = per_person[pid].S
    conf = per_person[pid].conf
    commit = per_person[pid].commit

    p_driver = SUM_over_v( wP[v] * S[v] )
    m_driver = SUM_over_v( wM[v] * S[v] )

    # Logistic squashing to 0..1
    p_term = sigmoid(p_driver)
    m_term = sigmoid(m_driver)

    # Delta parameters (scaled by confidence + commitment)
    deltaP_param = kP * p_term * conf * commit
    multM_param  = 1.0 + (kM * m_term * conf * commit)

    per_person[pid].deltaP_param = deltaP_param
    per_person[pid].multM_param  = multM_param

  # -------------------------
  # 3) Monte Carlo simulation of marginal value (leave-one-out)
  # -------------------------
  R = 20000  # tune for speed/precision
  results = init_stats_containers(eligible_people)

  FOR r IN 1..R:

    # 3a) sample baseline
    P0 = sample_beta(outcomes.baseline.p_success_beta.alpha, outcomes.baseline.p_success_beta.beta)
    Vs = sample_lognormal(outcomes.baseline.success_value_lognormal.mu, outcomes.baseline.success_value_lognormal.sigma)
    Vf = sample_fail_value(outcomes.baseline.fail_value)

    # 3b) full team outcome
    deltaP_total = 0
    multM_total = 1.0

    FOR pid IN per_person:
      deltaP_total += per_person[pid].deltaP_param
      multM_total  *= per_person[pid].multM_param

    # optional synergy (MVP: skip or simple pairwise)
    IF outcomes.calibration.synergy.enabled:
      (deltaP_synergy, multM_synergy) = compute_synergy(per_person, outcomes.calibration.synergy.k_gamma)
    ELSE:
      deltaP_synergy = 0
      multM_synergy = 1.0

    P_team = clamp(P0 + deltaP_total + deltaP_synergy, 0.0, 1.0)
    Vs_team = Vs * multM_total * multM_synergy
    EV_team = (P_team * Vs_team) + ((1 - P_team) * Vf)

    # 3c) marginal contribution per person via leave-one-out
    FOR pid IN per_person:

      P_minus = clamp(P0 + (deltaP_total - per_person[pid].deltaP_param) + deltaP_synergy, 0.0, 1.0)
      Vs_minus = Vs * (multM_total / per_person[pid].multM_param) * multM_synergy
      EV_minus = (P_minus * Vs_minus) + ((1 - P_minus) * Vf)

      dEV = EV_team - EV_minus
      results[pid].accumulate(dEV)

  FOR pid IN results:
    results[pid].mean_dEV = results[pid].mean()
    results[pid].p10 = results[pid].percentile(10)
    results[pid].p50 = results[pid].percentile(50)
    results[pid].p90 = results[pid].percentile(90)
    results[pid].stdev = results[pid].stdev()

  # -------------------------
  # 4) Smoothing + confidence gating + risk adjustment
  # -------------------------
  lambda = policy.smoothing.lambda
  conf_threshold = policy.confidence_gating.conf_threshold
  min_effective_conf = policy.confidence_gating.min_effective_conf

  effective_value = dict()

  FOR person IN eligible_people:
    pid = person.person_id
    mv_raw = results[pid].mean_dEV

    mv_prev = prev_state.prev_smoothed_marginal_values[pid] OR mv_raw
    mv_smooth = lambda * mv_raw + (1 - lambda) * mv_prev

    conf_i = per_person[pid].conf
    conf_gate = clamp(conf_i / conf_threshold, min_effective_conf, 1.0)

    # RiskAdj (MVP): penalize high-variance contributions slightly
    # (Production: explicit downside contribution, bus factor, dependency graph)
    risk_adj = 1.0 / (1.0 + 0.1 * results[pid].stdev_normalized())

    effective_value[pid] = max(0, mv_smooth * conf_gate * risk_adj)

  # -------------------------

  # -------------------------
  shares_raw = normalize_to_sum1(effective_value over eligible pids)

  # apply per-person floor/ceiling (if present), then renormalize
  shares_bounded = apply_floor_ceiling(shares_raw, eligible_people)
  shares_bounded = renormalize(shares_bounded)

  shares_prev = prev_state.prev_value_shares OR shares_bounded

    down = policy.change_caps.max_share_down  # e.g., 0.30 means -30% relative

    target = shares_bounded[pid]
    prev = shares_prev[pid] OR target

    max_up = prev * (1 + up)
    max_down = prev * (1 - down)

    shares_capped[pid] = clamp(target, max_down, max_up)

  shares_final = renormalize(shares_capped)

  # -------------------------
  # 6) Allocate variable compensation (and optional equity earn-out)
  # -------------------------
  pool_amt = policy.variable_pool.pool_amount_quarter
  allocations = []

  FOR person IN eligible_people:
    pid = person.person_id
    variable_pay = pool_amt * shares_final[pid]

    allocations.append({
      "person_id": pid,
      "variable_pay_quarter": variable_pay,
      "value_share": shares_final[pid],
      "expected_marginal_value": results[pid].mean_dEV,
      "p10": results[pid].p10,
      "p50": results[pid].p50,
      "p90": results[pid].p90
    })

  IF policy.equity_earnout.enabled:
    equity_pool = policy.equity_earnout.earnout_pool_equity_pct
    allocations = allocate_equity_earnout(allocations, equity_pool, policy.equity_earnout.max_earnout_per_person_pct)

  # -------------------------
  # 7) Build audit report (explainability)
  # -------------------------
  audit = {
    "cycle_id": cycle.cycle_id,
    "context_id": payload.context.context_id,
    "policy_id": policy.policy_id,
    "pool_amount_quarter": pool_amt,
    "method": {
      "simulation_runs": R,
      "baseline": outcomes.baseline,
      "calibration": outcomes.calibration,
      "smoothing": policy.smoothing,
      "confidence_gating": policy.confidence_gating,
      "change_caps": policy.change_caps
    },
    "per_person": []
  }

  FOR person IN eligible_people:
    pid = person.person_id
    drivers = top_vectors_by_contribution_proxy(per_person[pid].S, context_weights)

    audit.per_person.append({
      "person_id": pid,
      "assessment_id": per_person[pid].assessment_id,

      "confidence": per_person[pid].conf,
      "engagement_fte": person.engagement.fte_fraction,
      "marginal_value_stats": {
        "mean": results[pid].mean_dEV,

        "p50": results[pid].p50,
        "p90": results[pid].p90,
        "stdev": results[pid].stdev
      },
      "effective_value_used": effective_value[pid],
      "value_share_final": shares_final[pid],
      "variable_pay_quarter": pool_amt * shares_final[pid],
      "top_drivers": drivers,
      "bias_flags": get_bias_flags(pid, payload.assessments)
    })

  # -------------------------
  # 8) Output state for next cycle
  # -------------------------
  next_state = {
    "prev_value_shares": shares_final,
    "prev_smoothed_marginal_values": { pid: effective_value[pid] for pid in shares_final }
  }

  RETURN {
    "allocations": allocations,
    "audit_report": audit,
    "next_cycle_state": next_state
  }
