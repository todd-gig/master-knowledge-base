# Objective
Turn exception logs into operational intelligence.

# Key Metrics
- exception count by class
- exception rate by decision type
- exception rate by prompt version
- exception rate by architecture
- mean confidence by exception class
- human escalation frequency

# Canonical Outcome
Repeated exceptions are backlog for future deterministic automation.
