# Python Runtime Scaffold

This folder now includes:
- weighted routing,
- runtime payload validation,
- SQLite or Postgres audit logging,
- exception analytics,
- codification candidate scoring,
- migration runner,
- prompt release diffing and approvals,
- benchmark fixture utilities.

## Files
- `router.py` - weighted routing logic
- `validation.py` - lightweight schema validation
- `audit_store.py` - SQLite / Postgres persistence
- `analytics.py` - exception summaries and codification scoring
- `runtime.py` - end-to-end validated decision processing
- `migrate.py` - applies ordered SQLite migrations
- `release_manager.py` - prompt diffing and approval logic
- `benchmark.py` - benchmark fixture loader and mode comparison
