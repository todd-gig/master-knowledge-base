# Overview
This package is a full human + machine decision-routing system.

It now includes:
- human-readable operating policy in Markdown,
- machine-readable schemas and configs,
- runtime JSON schema validation,
- SQLite or Postgres audit logging,
- exception analytics,
- codification candidate scoring,
- automated tests,
- CI workflow,
- prompt/version registry,
- Alembic-style SQL migrations,
- dashboard-ready SQL views,
- prompt diffing and release approvals,
- benchmark fixtures for Claude vs Python cost/quality comparisons.
