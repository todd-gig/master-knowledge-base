# Architecture

## Logical architecture

### 1. Source layer
- transcript / chat thread
- packaged prompt files
- project logic notes

### 2. Extraction layer
- identify goals
- identify capabilities
- identify skills
- identify structure and supervision logic
- identify negotiation activities

### 3. Codification layer
- Markdown knowledge assets
- machine-readable JSON registries
- starter Python exporters/parsers
- test harness

### 4. Runtime layer
- prompt routing
- file selection
- output templating
- automation vs human boundary enforcement

### 5. Governance layer
- approval paths
- supervision requirements
- qualification layers
- auditability through transcript preservation

## Design implication

This is a **prompt-first knowledge operating system** with machine-readable sidecars.
