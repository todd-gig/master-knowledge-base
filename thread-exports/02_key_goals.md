# Key Goals

## Explicit goals

1. Itemize all skills needed within the project.
2. Build a C-suite-down master skills matrix.
3. Use fixed span rules:
   - each executive: 7 direct reports
   - each subordinate layer: 3-5 or up to 3
4. Make the system granular enough to identify:
   - skills
   - talents
   - experience
   - qualifications
   - supervision requirements
5. Add CMO, CRO, Chief Environment Officer, and Chief Policy Officer / Chief Political Advisor.
6. Create a Claude project prompt to apply this logic.
7. Create companion Markdown files.
8. Package them in a zip bundle.
9. Add a routing index for companion files.

## Inferred goals

1. Build a reusable governance layer for future enterprise design.
2. Turn org design into a machine-readable decision system.
3. Create transcript-backed assets for future training and automation.
4. Reduce ambiguity in skill assignment and negotiation planning.
5. Standardize human-in-the-loop control points.
