# Foundational Principles
1. **Every recurring decision is a unit economics problem.**
2. **Deterministic execution is cheaper than LLM reasoning.**
3. **Stable logic should be codified.**
4. **Ambiguity should be resolved before hard automation.**
5. **Repeated Claude use on stable work is a migration signal.**
6. **Prompt logic without runtime controls is not enough.**
7. **Markdown is the human governance layer.**
8. **Code is the enforcement and execution layer.**
9. **The winning system is a feedback loop:**
   - Claude interprets,
   - Python executes where possible,
   - results are logged,
   - patterns are measured,
   - codification expands,
   - recurring cost drops.

# Economic Rule
**Preferred execution layer = highest expected net value at scale.**

# Canonical Flywheel
```md
Claude -> discovers logic -> Python codifies -> cost drops -> scale increases -> ROI compounds
```
