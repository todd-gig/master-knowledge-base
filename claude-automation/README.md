# Objective
This package exports the complete logic developed in this thread into a reusable **Markdown + code bundle** suitable for:
- Claude project instructions,
- repo-level `CLAUDE.md`,
- Obsidian vault storage,
- runtime execution,
- audit logging,
- prompt governance,
- deployment planning,
- codification from Claude to Python.

# Package Intent
The intelligence in this thread is centered on one operating law:

> **Claude discovers logic. Python scales it profitably.**

That law is operationalized as:
1. route ambiguous work to Claude,
2. route stable deterministic work to Python,
3. use Hybrid patterns during transition states,
4. log outcomes,
5. identify codification candidates,
6. migrate repeatable logic out of Claude and into software.
