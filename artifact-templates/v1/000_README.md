# Artifact Template Pack

This folder contains one markdown template per product or service.

## Intended Use
Use these files in Claude as reusable templates for future interactions where the same artifact must be created for a different:
- brand
- business
- environment
- use case
- why
- goal
- objectives
- description
- timing
- channel(s)

## Template Rule
Each file is intentionally input-heavy and output-ready. Replace placeholders, preserve structure, and complete all required sections before generation.

## File Convention
`NNN_slug.md`

## Core Required Inputs
- Brand
- Business
- Environment
- Use Case
- Why
- Goal
- Objectives
- What
- Description
- When
- Where / Channels
- Audience
- CTA
- Constraints
- Dependencies
- Approval Requirements
